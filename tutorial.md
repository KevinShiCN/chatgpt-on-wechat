## 1. 文字对话

### 功能说明
直接与 AI 进行文字对话，获取问题解答、文案撰写、信息查询等服务。

### 操作视频
https://drive.weixin.qq.com/s?k=AMYArQc3AAc1BNwZfcAIUABwY3AMM

### 如何调用
在对话框中直接输入您想问的问题，然后发送即可。

### 使用示例
```
用户：帮我写一份会议纪要模板
用户：解释一下什么是OKR
用户：翻译这段话成英文：今天天气很好
```

### 小技巧
- 问题描述越清晰，回答越准确
- 可以进行多轮对话，AI 会记住上下文
- 复杂问题可以分步骤提问

## 2. 图片+文字问题

### 功能说明
发送图片配合文字描述，让 AI 分析图片内容、识别文字、解读图表等。

### 操作视频
https://drive.weixin.qq.com/s?k=AMYArQc3AAc1BNwZfcAIUABwY3AMM

### 如何调用
1. 先输入您的问题文字
2. 在 **20秒内** 发送相关图片
3. 等待 AI 分析回复

### 使用示例
```
用户：这张图片里有什么内容？
用户：[发送图片]

用户：帮我识别这张图片中的文字
用户：[发送图片]

用户：分析一下这个数据图表
用户：[发送图片]
```

### 注意事项
- **重要**：文本和图片发送间隔不能超过 **20秒**
- 建议在电脑端使用，方便准备好所有内容后一起发送
- 支持多张图片，但需在时间窗口内连续发送

## 3. 文生图

### 功能说明
根据文字描述生成 AI 图片，支持多种风格和比例。

### 操作视频
https://drive.weixin.qq.com/s?k=AMYArQc3AAc1HA7EZJAIUABwY3AMM

### 如何调用
在对话框中以 **【生图】** 或 **【画】** 开头，后面跟上图片描述。

### 命令格式
```
生图 [比例(可选)] [图片描述]
画 [比例(可选)] [图片描述]
```

### 支持的图片比例
| 比例 | 适用场景 |
|------|---------|
| 1:1 | 头像、图标 |
| 4:3 | 传统照片 |
| 3:4 | 竖版海报 |
| 16:9 | 横版封面、PPT配图 |
| 9:16 | 手机壁纸、短视频封面 |
| 2:3 / 3:2 | 杂志封面 |
| 4:5 / 5:4 | 社交媒体图片 |

### 使用示例
```
生图 一只可爱的橘猫在阳光下打盹
画 16:9 科技感十足的未来城市
生图 4:3 中国风水墨山水画
画 一个穿着西装的商务人士
```

### 小技巧
- 描述越详细，生成效果越好
- 可以指定风格：写实、卡通、水彩、油画等
- 可以描述光线、角度、氛围等细节
- 生成时间约 1-2 分钟，请耐心等待

## 4. 改图（图生图）

### 功能说明
基于参考图片进行二次创作，可以修改风格、添加元素、调整内容等。

### 操作视频
https://drive.weixin.qq.com/s?k=AMYArQc3AAc1HA7EZJAIUABwY3AMM

### 如何调用

**方式一：先发命令，后发图片**
```
用户：生图 把这张照片变成动漫风格
[10秒内发送参考图片]
用户：[发送图片]
```

**方式二：先发图片，后发命令**
```
用户：[发送参考图片]
[10秒内发送命令]
用户：生图 把背景换成海边
```

### 使用示例
```
生图 把这张人物照片变成赛博朋克风格
[发送人物照片]

[发送产品图片]
画 给这个产品添加科技感光效

生图 16:9 把这张风景照变成油画风格
[发送风景照片]
```

### 注意事项
- 图片和命令的发送间隔不要超过 **10秒**
- 支持发送多张参考图片（每张图片后有20秒等待时间）
- 系统会提示"收到了X张图片，以及提示词：XXX"
- 生成时间约 1-2 分钟

## 5. 批量抽卡

### 功能说明
一次性生成多张图片，适合需要多个创意方案或批量素材的场景。

### 如何调用
使用 **【抽卡】** 前缀 + 数量 + 图片描述。

### 命令格式
```
抽卡[数量] [比例(可选)] [图片描述]
```

### 使用示例
```
抽卡3 可爱的卡通猫咪头像
抽卡5 16:9 科幻风格的宇宙飞船
抽卡10 简约风格的logo设计
抽卡 美丽的风景画（默认3张）
```

### 配合改图使用
```
[发送参考图片]
抽卡5 把这张照片变成不同的艺术风格
```

### 执行过程示例
```
用户：抽卡3 一只可爱的小狗

系统：开始抽卡，共3张(文生图模式)，请耐心等待...
      提示词：一只可爱的小狗

系统：[图片1]
系统：第1/3张生成完成

系统：[图片2]
系统：第2/3张生成完成

系统：[图片3]
系统：第3/3张生成完成

系统：抽卡完成！共生成3张图片
```

### 使用限制
- 单次最多生成 **20张** 图片
- 默认生成 **3张** 图片（不指定数量时）
- 最少生成 **1张** 图片

### 小技巧
- 适合需要多个创意方案时使用
- 可以从多张结果中挑选最满意的
- 生成过程中会显示进度，方便了解状态
- 即使部分图片生成失败，也会继续生成剩余图片

## 6. 使用小技巧

### 提高对话质量
1. **明确需求**：清晰描述您想要什么，避免模糊表达
2. **提供背景**：告诉 AI 使用场景和目的
3. **分步提问**：复杂问题拆分成多个小问题
4. **追问细节**：对回答不满意可以要求补充或修改

### 提高图片质量
1. **详细描述**：包含主体、风格、颜色、光线、氛围等
2. **指定风格**：写实、卡通、水彩、油画、3D渲染等
3. **选择比例**：根据用途选择合适的图片比例
4. **多次尝试**：使用抽卡功能获取多个方案

### 高效使用建议
1. **电脑端操作**：涉及图片时建议使用电脑端，方便准备素材
2. **提前准备**：发送图片前先准备好所有素材
3. **注意时间**：图片和文字的发送间隔不要超过规定时间
4. **耐心等待**：图片生成需要1-2分钟，请勿重复发送

## 7. 常见问题

### Q: 为什么我的图片没有被识别？
A: 请确保图片在文字发送后的 **20秒内** 发出。建议在电脑端操作，准备好后一起发送。

### Q: 生图需要等多久？
A: 通常需要 **1-2分钟**，系统会每60秒发送进度提醒。请耐心等待，不要重复发送命令。

### Q: 抽卡最多能生成多少张？
A: 单次最多 **20张**。如果需要更多，可以分多次抽卡。

### Q: 改图时图片发送顺序有要求吗？
A: 没有严格要求。可以先发命令后发图片，也可以先发图片后发命令，但间隔不要超过 **10秒**。

### Q: 生成的图片不满意怎么办？
A: 可以尝试：
- 修改描述词，增加更多细节
- 使用抽卡功能生成多张选择
- 调整图片比例
- 指定不同的风格

### Q: 支持哪些图片格式？
A: 支持常见的图片格式，包括 JPG、PNG 等。
